TDD in a nutshell

  • Add a test (but not the actual code)
  • Run all tests and make sure the new one fails
  • Write some code to pass the new test
  • Run all tests and make sure all tests pass
  • (Optionally) Refactor the code
  • then run all tests again and make sure all tests pass
  • Repeat
